Inspecting the DOM With Chrome

PROBLEM 1:

Navigate to https://www.google.com/ and open the dev tools. What is the element type of the Google logo?

Ans: The element type is an 'img' element.

PROBLEM 2:

Working off of problem 1, what attributes does this element have?

Ans: It has a class, alt, height, width, title, id, data-csiid and data-atf attributes.

PROBLEM 3

What's the special element where the user will input their Google search?

Ans: The special element is called 'textarea'.

PROBLEM 4

What's the 3rd child of the form element?

Ans: A style tag

PROBLEM 5 (DEBUGGING)

Take a look at the window object in the console. (hint: open the console and devtools and type 'window' and hit enter).

Which key on the window object shows the language of your browser window?
